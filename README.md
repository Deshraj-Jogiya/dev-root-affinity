# 🧠 Dev-Root-Affinity: My Inception & Lifelong Developer Journey

Welcome to **Dev-Root-Affinity**, a personal curation of learning logs and "Today I Learned" (TIL) entries capturing my software engineering journey and lifelong dedication to technology since **January 2018** (the start of my Computer Engineering B.Tech degree).

This repository serves as my **"Root Node"**—the origin point of my career. It chronicles my technical evolution from learning pointers and OOP structures, through remote data projects during the COVID-19 lockdown, to advanced ML/Data engineering research at Arizona State University (ASU) and industry solutions.

---

## 📅 Roadmap of Technical Growth & Milestones

* **2018 – 2019 (B.Tech Foundation):** Focus on C/C++ programming, object-oriented concepts, basic data structures, and SQL aggregations.
* **2020 – 2021 (Lockdown & Graduation):** Deepening focus on Python scripting, Pandas data analysis, REST APIs, and team collaboration workflows during the COVID-19 quarantine.
* **2022 – 2024 (ASU Master's Program):** Deep-dive into Cloud Systems (AWS), advanced database engineering (Snowflake, MongoDB), and Machine Learning architectures (Scikit-learn, TensorFlow).
* **2025 – 2026 (ML/Data Industry Experience):** Applied AI, LLM prompting strategies, fine-tuning API parameters, and serverless automation frameworks (FastAPI + GitHub Actions).
